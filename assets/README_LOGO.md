# ZENXHAT Logo Assets

## Available Formats

### 1. **zenx-logo.txt** (ASCII Art)
- Terminal-friendly format
- Use untuk console/CLI display
- Perfect untuk README header

```bash
cat assets/zenx-logo.txt
```

### 2. **zenx-logo.svg** (Vector Graphics)
- Scalable format
- Use untuk web/documentation
- Modern cyberpunk design dengan glow effect
- Colors: Neon Green (#00ff88), Cyan (#00ccff), Purple (#8800ff)

### 3. **Colors Reference**
```
Primary Color:    #00ff88 (Neon Green)
Secondary Color:  #00ccff (Cyan)
Accent Color:     #8800ff (Purple)
Background:       #0a0e27 (Dark Blue)
```

## Usage in README

```markdown
![ZENXHAT Logo](./assets/zenx-logo.svg)

or

```
cat assets/zenx-logo.txt
```
```

## Logo Specifications

- **Design**: Modern cyberpunk aesthetic
- **Style**: Geometric with "Z" and "X" intersection
- **Theme**: Ethical hacking, OSINT, information gathering
- **Glow Effect**: Neon cyberpunk style with filter effects

## Customization

To modify colors in SVG:
1. Edit the `<linearGradient>` section
2. Change `stop-color` values
3. Adjust `stroke-width` for thickness
4. Modify filter `stdDeviation` for glow intensity
